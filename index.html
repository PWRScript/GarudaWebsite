Garuda Linux

O Garuda Linux é uma distribuição linux baseada em Arch, cujo seu objetivo é permitir a utilizadores domésticos instalar e usar com facilidade uma distribuição Arch Linux, mantendo ao mesmo tempo um design/aparência e animações bonitas e futuristicas

As suas principais caracteristicas:

-> Rolling Release (Este é facilmente atualizado quando o sistema base (Arch) recebe novas atualizações)

-> Facilidade de Instalação/Uso (Quase todas as ferramentas que requeriam o uso de CLI podem ser configuradas/usadas usando uma Interface Gráfica)

-> Optimizações que melhoram a utilização da Memória RAM/CPU/Disco Rígido (tornando o sistema mais responsivo)

-> Troca rápida de kernels e a sua instalação padrão tem um kernel muito optimizado para performace

-> Sistema de Backups/Restauração instantâneo usando BTRFS e possiblidade de restaurar o sistema no arranque (GRUB), sem necessitar do disco de instalação
